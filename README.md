## 📋 Sobre a Atividade
A atividade consistiu na produção de uma resenha fundamentada na **Seção 7.4 (Microsserviços)** do material de Engenharia de Software Moderna. O trabalho aborda a transição do modelo monolítico para o distribuído, focando em:

* **Gargalo Arquitetural:** Análise de como sistemas monolíticos travam o ciclo de entregas ágeis devido ao acoplamento em tempo de execução.
* **Isolamento de Processos:** Estudo sobre como a separação de módulos em processos independentes no Sistema Operacional garante o desacoplamento.
* **Escalabilidade e Resiliência:** Discussão sobre escalabilidade horizontal granular e tolerância a falhas parciais.
* **Lei de Conway:** Reflexão sobre a relação entre a estrutura das equipes de desenvolvimento e a arquitetura do software produzido.

A resenha também estabelece uma ponte teórica com projetos acadêmicos realizados anteriormente, avaliando como a adoção dessas práticas otimizaria o desenvolvimento e a manutenção de aplicações reais.
